# Shahriar Hosen — Personal Brand Website

Premium static HTML/CSS/JS website. No build step. No framework.

## Tech
- HTML5, Bootstrap 5.3 (CDN), Bootstrap Icons, Google Fonts
- Vanilla JavaScript
- Mobile-first responsive
- Dark / Light mode (persisted in localStorage)
- SEO + OpenGraph + favicon

## Run locally
Just open `index.html` in your browser, or serve the folder:

```
python3 -m http.server 8080
```

Then visit http://localhost:8080
